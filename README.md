# profile.io

A modern portfolio website built with **Vite** and **Tailwind CSS v4**.

## Features
- **Vite 6**: Fast development server and optimized builds.
- **Tailwind CSS v4**: Modern, utility-first styling with zero-runtime overhead.
- **Modern UI**: Clean design with Inter font and a refined color palette.
- **Responsive**: Fully responsive layouts using modern CSS Grid and Flexbox.
- **ES Modules**: Modern JavaScript for interactive elements.

## Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm

### Installation
```bash
npm install
```

### Development
Start the development server with HMR:
```bash
npm run dev
```

### Build
Build for production:
```bash
npm run build
```
